## Background
This project is a simple web-based text-to-speech application developed as part of a mini-project.  
The application allows users to type text and convert it into speech using the Web Speech API supported by modern browsers.

## Features
- Converts typed text into speech
- Voice selection from available browser-supported voices
- Displays status messages during speaking
- Simple and clear user interface
- Works directly in the browser without backend or install

## Installation

### Online
Visit the live demo link : https://dhanya-akshaya.github.io/text-to-speech-website/

### Run Locally
1. Download or clone the project  
   `git clone https://github.com/dhanya-akshaya/text-to-speech-website.git`
2. Open the project folder
3. Open the `index.html` file in any modern web browser

## Usage
1. Enter text in the text box
2. Select a voice from the dropdown (optional)
3. Click the **Speak** button
4. The browser will speak the text aloud
